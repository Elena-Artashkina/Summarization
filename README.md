 # Смысл.txt

Веб-сервис для анализа и обработки текстовых данных с использованием технологий машинного обучения.

## 🚀 Возможности

- Анализ текстов, путем формирования статистических показателей, облака слов и анализа частей речи
- СММ помощник - проверка текстов на корректность для ведения социальных сетей
- Суммаризация - генерация кратного содержания с использованием модели машинного обучения

## 🛠️ Технологии

- Python, NodeJS
- Llama-cpp
- FastAPI
- React
- PostgreSQL
- Nltk
